## Getting Started

Include the starter project in one of your spring boot applications to get started:

```xml

<dependency>
    <groupId>de.gcoding.boot</groupId>
    <artifactId>database-spring-boot-starter</artifactId>
    <version>1.0.0</version>
</dependency>
```
